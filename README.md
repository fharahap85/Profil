# AFH Portfolio

Personal portfolio website built with [Next.js](https://nextjs.org/) 14, [TypeScript](https://www.typescriptlang.org/), and [Tailwind CSS](https://tailwindcss.com/).

## Tech Stack

- **Framework:** Next.js 14 (App Router, static export)
- **Language:** TypeScript, React 18
- **Styling:** Tailwind CSS 3.4 with glassmorphism and bento grid design
- **Icons:** Lucide React
- **Deployment:** GitHub Pages (static export)

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
pnpm build
```

Static output in `out/`.
